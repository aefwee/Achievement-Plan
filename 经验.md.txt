1。 不要用国产浏览器， 很卡很占内存，对开发不友好， 用chrome浏览器和360浏览器
2.   能直接访问域名就直接访问， 不要通过百度， 这样能省很多事
3.    下载chrome太慢了， 就放弃吧 ， 抓住重点
4.   通过专业的方式卸载软件
5.   Windows电脑不要在桌面或者C盘任何地方建立开发环境（因为win有超级管理员和防火墙问题会导致你的程序有时候会有奇怪的bug）
6.   作程序员肯定要把 ·文件扩展名· 和 隐藏的项目· 这2个选项打开
7.   如何将文件上传到github进行保存：
         第一步：在github建立仓库
         第二步：利用git将文件上传到github仓库
            大致流程： git clone 仓库地址
                              git pull(从远程仓库拉取文件)
                              git add .
                              git commit -m ""
                               git push -u origin master(将本地文件推送到远程仓库)
